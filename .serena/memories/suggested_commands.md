# Suggested Commands

## Development
```bash
# Start Next.js development server
pnpm dev

# Build Next.js for production
pnpm build

# Start production server
pnpm start

# Build Chrome extension (development)
pnpm dev:extension

# Build Chrome extension (production)
pnpm build:extension

# Package extension
pnpm package

# Fetch latest download
pnpm fetch-latest
```

## Testing
```bash
# Run the development server and cancel it quickly
pnpm dev
# (then cancel with Ctrl+C)
```

## Git Operations
- Use conventional commit format: `chore(scope): message`
- No AI attribution in commits
- Never modify git config

## System Commands (macOS Darwin)
- Standard Unix commands available: `ls`, `cd`, `grep`, `find`, etc.
- Use `open` for opening files/URLs
